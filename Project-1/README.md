# DecodeLabs Cloud Computing Project 1

## Project Description

This project demonstrates hosting a static portfolio website using Amazon S3 Static Website Hosting.

## Technologies Used

- HTML5
- CSS3
- AWS S3
- Git
- GitHub

## Features

- Responsive Design
- About Section
- Skills
- Projects
- Education
- Contact Information

## Deployment Steps

1. Create an S3 Bucket
2. Disable Block Public Access
3. Upload all project files
4. Enable Static Website Hosting
5. Apply Bucket Policy
6. Open Website Endpoint
